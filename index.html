Anwendungshilfe
  • Im Feld "AC-Nummer des Kopfsatzes" muss der übergeordnete Datensatz eingegeben werden, wobei das Präfix "AC" ausgelassen werden kann
  • Der Wert im Dropdown beeinflusst nur die Information, ob für diese Institution Holdings vorhanden sind, die nicht vom Publishing unterdrückt wurden ("Lokalbestand")
  • Das Laden vieler Daten kann wegen der SRU Calls dauern, z. B. für 1000 Datensätze ca. 50 Sekunden
  • Die Daten werden immer aus der Network Zone geholt
  • Die Daten werden nach Zählung ("773$$q bzw. 830$$v") sortiert geladen und können per Klick auf die Tabellenüberschrift neu sortiert werden, immer nur nach exakt einer Spalte
  • Nur die Spalte "773$$q bzw. 830$$v" hat eine spezielle Sortierung, alle anderen Spalten sortieren streng alphabetisch (zum Beispiel sortiert "Band 10: Gedichte" vor "Band 2: Prosa")
  • Titel werden annähernd in ISBD Interpunktion dargestellt, so weit das im Rahmen einer Automatisierung möglich ist
  • Über "CSV Download" kann die Tabelle heruntergeladen werden (Semikolon-getrennt, UTF8)
