Anwendungshilfe
  • Im Feld "AC-Nummer des Kopfsatzes" muss der übergeordnete Datensatz eingegeben werden
  • Die Daten werden immer aus der Network Zone geholt
  • Die Eingabe der Institution beeinflusst nur die Information, ob für diese Holdings vorhanden sind, die nicht vom Publishing unterdrückt sind ("Lokalbestand")
  • Das Laden vieler Daten kann wegen der SRU Calls dauern, z. B. für 1000 Datensätze ca. 50 Sekunden
  • Die Daten werden unsortiert geladen, aber es ist eine Sortierung möglich durch Klick auf die Tabellenüberschrift
  • Titel werden annähernd in ISBD Interpunktion dargestellt, aber nur so weit das im Rahmen einer Automatisierung überhaupt möglich ist
